Performance Based Appraisal System (PBAS)

Performance Based Appraisal System (PBAS) for the DAVV, indore University faculties

New

This is my last internship project with the univercity of Indore as a Full Stack web developer in the 6th sem of MCA. It is a Performance Based Appraisal System (PBAS) for the DAVV, indore University faculties, specially for the UTD Department IIPS. PBAS is a Annual Self- Assessment for the Performance Based Appraisal System (PBAS) Session/ Year which is to be to be completed and submitted at the end of each academic year. I was working on its automation. We made a system by which each faculty submit their records time to time and at the end of each session they can easily generate a PDF Report on that session, along with he/she can view their previous records also. It is a of responsive Bootstrap Official College Website of IIPS-DAVV, a Department of DAVV, indore, and is under the development phase, will be live soon. You can check the code here on githubwww.iips.edu.in
